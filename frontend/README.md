# 🌐 AutoTesteAI - Frontend

Dashboard premium para gerenciamento de automação de testes exploratórios.

## 🚀 Tecnologias

- **React 19**: Framework de UI.
- **Vite 8**: Build tool ultra-rápida.
- **Socket.io Client**: Recebimento de logs em tempo real do backend.
- **Framer Motion**: Animações de interface e transições.
- **Lucide Icons**: Conjunto de ícones premium.

## 📦 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Configure a URL do backend no arquivo `App.jsx` (padrão: `http://localhost:3002`).

3. Rode em modo desenvolvimento:
   ```bash
   npm run dev
   ```

Acesse: **[http://localhost:5173](http://localhost:5173)**

---

> [!NOTE]
> Este painel depende da execução simultânea do servidor backend localizado em `/server`.
